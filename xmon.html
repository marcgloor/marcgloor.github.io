xmon - Minimalistic Lightweight System Performance Monitor
by Marc Gloor

Abstract

Xmon is a very puristic, fast and minimalistic lightweight system performance monitor for Linux, purely written in C and linked only against glibc and X11/Xlib. No additional heavyweight window manager bloat included, I kept it lean.

It visualizes real-time system load, memory usage, and CPU activity. The tool operates by reading data directly from the Linux process filesystem (/proc), which must be mounted for the application to function. Read the manpage or the README to learn more about its full functionality.

  * CPU Usage [CPU]: Total processor utilization (central parallel unit usage)
  * Memory [MEM]: Current RAM consumption (used memory)
  * System Load [LOAD]: Standard Linux load average (load average)

Xmon was the result of an educational rainy day learning hardcore Xlib coding back in 1998 - consequently, the codebase is in a beta state, largely unoptimized, and may contain bugs. Early version bugs have all been fixed, such as e.g. the memory overflows and necessary graph reset every x cycles.

Early 2026 with release 1.0.1, a major code modernization and new features have been introduced in order to brush-up the code and make it work with any modern Linux kernels and X11 environments, tested on a most recent Debian SID (Feb 26) system. While not natively portable to other Unix derivatives like Solaris or AIX, it is compatible with any hardware (such as Sun SPARC or IBM's RISC 6000 cpu's running a full fletched Linux environment. I am releasing the source code under the GNU GPL v3 in the hope that it's useful for you. In case you find bugs, let me know.

Installation

Source package: xmon-1.0.1-src.tar.gz
Download, extract and follow the README instructions to compile and install xmon.

Precompiled binary package: xmon-1.0.1-statbin.tar.gz
Download Linux x86-64 ELF (amd64) binary distribution that runs out of the box.

Debian binary package for amd64 architecture: xmon_1.0.1_amd64.deb
Download and install using your package manager e.g. dpkg or apt, in runs out of the box.
