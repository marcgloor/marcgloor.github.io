xdict - a scientific fuzzylogic algorithm implementation
by Marc O. Gloor

Abstract

xdict is a graphical english/german dictionary running on the Linux operating system, it was written using the Gtk+ API. For further informations read the manpage.
Xdict-icd is an interesting derivative package which was written by Thomas Egglseder, the program searches for medical ICD-10-Keys. 

The german Magazine Linux-User has also presented the tool in 2005, click here to see the article.
